Level 1

This level contains very basic calls to the API it utilizes the most basic methods to make use of the in memory data store

GET
/items
Retrieve all items from the in-memory data store
POST
/post-items

PUT
/items/:id
Update an existing item by ID. Check the /items endpoint to get ids currently available.
